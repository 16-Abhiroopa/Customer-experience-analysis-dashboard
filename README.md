Customer Experience Analysis Dashboard using Microsoft Excel
Project Title

Customer Experience Analysis Dashboard
Project Overview: 

The Customer Experience Analysis Dashboard is an interactive Excel dashboard developed to analyze customer support requests and customer satisfaction (CSAT). The dashboard helps businesses monitor service performance, identify customer issues, evaluate SLA compliance, and understand customer satisfaction across different support channels and call centers.The project demonstrates how Microsoft Excel can be used as a Business Intelligence tool through Pivot Tables, Pivot Charts, Slicers, and Dashboard Design.

Objective

The primary objective of this project is to:
Analyze customer support requests.Measure customer satisfaction (CSAT). Monitor SLA (Service Level Agreement) performance.Identify high-volume issue categories.
Compare support channels.Assist business stakeholders in making data-driven decisions.Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Conditional Formatting
Dashboard Design
Data Cleaning
Excel Functions
 
Dataset Description
The dataset contains customer support ticket information including:
Customer ID
Support Channel
Response Time
Call Center
State
Issue Category
Call Duration
Customer Satisfaction (CSAT)
SLA Status

Data Cleaning Process
Before creating the dashboard, the dataset was cleaned by:
Removing duplicate records.
Handling missing values.
Standardizing text values.
Correcting inconsistent formatting.
Verifying data accuracy.
Creating calculated fields where required.

Challenges Faced
During development:
Cleaning inconsistent data.
Handling blank values.
Designing an interactive dashboard layout.
Creating dynamic Pivot Tables.
Synchronizing slicers across multiple Pivot Charts.
Maintaining dashboard readability.
